[CSS 연습 중...]

CSS는 웹 페이지의 스타일을 꾸미기 위한 스타일 시트 언어입니다. HTML 요소의 모양, 레이아웃, 색상 등을 정의하여 웹 페이지의 시각적인 부분을 담당합니다. 즉, HTML이 웹 페이지의 뼈대를 구성한다면 CSS는 옷을 입히는 역할을 한다고 볼 수 있습니다. CSS의 주요 기능: 스타일 지정: 글꼴, 색상, 배경, 테두리, 간격 등 다양한 스타일 속성을 사용하여 웹 페이지 요소의 모양을 지정합니다. 레이아웃 구성: 웹 페이지의 요소들을 원하는 위치에 배치하고, 화면 크기에 따라 반응형으로 레이아웃을 구성합니다. 화면 표시 제어: 웹 페이지가 다양한 장치(데스크톱, 모바일, 프린터 등)에서 어떻게 표시될지 제어합니다.
